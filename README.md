# Yeti-Oracle
Yeti Oracle is an AI-powered Web3 gaming platform that combines fantasy storytelling, interactive mini-games, and blockchain technology into a unique user experience. Players connect their wallets, receive AI-generated predictions, complete quests, earn XP, unlock achievements, and climb leaderboards.
